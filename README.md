# misa

查看示例

[demo](https://misa-beige.now.sh/) |　[示例网站](https://misa-beige.now.sh/)

本项目使用　Ｗebpack & Sass 创建的静态杂志网站的响应式网页模板。探索中文网页的排版和实践。

```
git clone https://github.com/disism/misa.git && cd misa
```

```
npm install
```

```
# 开发环境
npm run dev

# 打包到 public 目录
npm run build
```

欢迎访问 [disism.com](https://disism.com) 获取更多内容
